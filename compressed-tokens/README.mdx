| Creation      | Solana    | Compressed         |
| :------------- | :--------------------- | :---------------------- |
| **Token Account** | 	~2,000,000 lamports          | **5,000** lamports     |

1. Compressed token accounts store token balance, owner, and other information of tokens like SPL and c-Tokens. 
2. Compressed token accounts are rent-free.
3. Any c-Token or SPL token can be compressed/decompressed at will.
<Info>
Wallets like Phantom and Backpack support compressed tokens. The UI does not distinguish between SPL and compressed tokens.
</Info>
