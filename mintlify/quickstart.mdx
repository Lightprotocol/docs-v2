## Local development

1. Install the Mintlify CLI:
   ```bash
   npm i -g mint
   ```

2. Navigate to the docs directory and start the preview server:
   ```bash
   mint dev
   ```

3. Open `http://localhost:3000` to preview your changes.

<Tip>The preview updates automatically as you edit files.</Tip>
