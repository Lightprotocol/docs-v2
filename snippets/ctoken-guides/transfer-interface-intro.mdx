      <td>**SPL token → cToken Account**</td>
      <td>
        <ul>
          <li>Transfers SPL tokens to cToken accounts</li>
          <li>SPL tokens are locked in interface PDA</li>
          <li>cTokens are minted to cToken account</li>
        </ul>
      </td>
    </tr>
    <tr>
      <td>**cToken → SPL Account**</td>
      <td>
        <ul>
          <li>Releases SPL tokens from interface PDA to SPL account</li>
          <li>Burns cTokens in source cToken account</li>
        </ul>
      </td>
    </tr>
  </tbody>
</table>

<Tip>
* For example, **SPL → cToken** can be used for transfers from Alice's SPL token account to her own cToken account.
* You can use this to **convert existing SPL tokens to cTokens** with **sponsored rent-exemption**. Learn more in the [core concepts to the cToken program](/compressed-token-program/ctoken/README).
</Tip>
