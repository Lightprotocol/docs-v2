<Tabs>
  <Tab title="npm">

  Install packages in your working directory:
    ```bash
    npm install @lightprotocol/stateless.js@beta \
                @lightprotocol/compressed-token@beta
    ```
Install the CLI globally:
    ```bash
    npm install -g @lightprotocol/zk-compression-cli@beta
    ```
  </Tab>
