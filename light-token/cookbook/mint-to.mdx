<Step>

### Build Account Infos and CPI the Light Token Program

Use `invoke` for external signers or `invoke_signed` when the authority is a PDA.

<Tabs>
<Tab title="invoke (External signer)">

```rust
use light_token::instruction::MintToCpi;

MintToCpi {
    mint: mint.clone(),
    destination: destination.clone(),
    amount,
    authority: authority.clone(),
    system_program: system_program.clone(),
    fee_payer: None,
    max_top_up: None,
}
.invoke()
```

</Tab>
<Tab title="invoke_signed (PDA authority)">

```rust
use light_token::instruction::MintToCpi;

let signer_seeds: &[&[u8]] = &[MINT_AUTHORITY_SEED, &[bump]];

MintToCpi {
    mint: mint.clone(),
    destination: destination.clone(),
    amount,
    authority: authority.clone(),
    system_program: system_program.clone(),
    fee_payer: None,
    max_top_up: None,
}
.invoke_signed(&[signer_seeds])
```

</Tab>
</Tabs>

<Info>
  `fee_payer` and `max_top_up` are optional fields to customize rent top-ups.
  Set to `None` to use defaults.
</Info>

</Step>
</Steps>
