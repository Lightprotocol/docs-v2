<Accordion title="Prerequisite: How Rent Sponsorship works">
Rent sponsorship is a built-in feature of the Light SDK’s that sponsors rent-exemption for all account types to reduce creation cost.
This is dealt with under the hood in a way that doesn’t disrupt the UX of what your users are used to with SPL-token.

## How to sponsor rent top-ups and transaction fees

Set the `payer` parameter to the sponsor’s public key when calling `createTransferInterfaceInstructions` (TypeScript) or building a `TransferInterface` instruction (Rust). The sponsor pays SOL for rent top-ups and transaction fees while the user only signs to authorize the transfer.

<Info>
You can set the payer to any signing account on any Light Token transaction.
</Info>
