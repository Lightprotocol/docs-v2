## Key Points

* Like Token 2022, each extension adds specific state to an account. 
* Depending on the extension, it can be added at account creation or existing accounts can migrate.
* Both compressible and token metadata extension are compatible with Token 2022
* The compressible extension is *not limited to token accounts*
